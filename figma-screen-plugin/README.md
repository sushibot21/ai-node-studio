# Node Studio — Screen Builder for Figma

1. In Figma Desktop, open **Plugins → Development → Import plugin from manifest…**.
2. Select `manifest.json` in this folder.
3. Run **Plugins → Development → Node Studio — Screen Builder**.
4. Paste a JSON screen specification and choose **Create screen**.

The plugin creates an editable native Figma frame, text, cards, and CTA buttons. It does not use or require an API token.
